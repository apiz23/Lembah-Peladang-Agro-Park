# Lembah Peladang Agro Park Revamp

This repository contains the revamped website for Lembah Peladang Agro Park, built using TypeScript, React, Shadcn UI, Node.js with Express, and MongoDB.

## Introduction

Lembah Peladang Agro Park Revamp is a web application aimed at providing an enhanced user experience for visitors and enthusiasts of Lembah Peladang Agro Park. The application is built with modern web technologies, including TypeScript for type safety, React for the user interface, Shadcn UI for styling, Node.js with Express for the backend, and MongoDB for the database.

## Features

- Responsive and user-friendly interface
- Dynamic content updates
- Integration with MongoDB for data storage
- Utilizes Shadcn UI components for a visually appealing design

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- Node.js: [Download Node.js](https://nodejs.org/)
- MongoDB: [Download MongoDB](https://www.mongodb.com/try/download/community)

## Installation

1. Clone the repository:

   ```bash
   https://github.com/apiz23/Lembah-Peladang-Agro-Park.git

2. Navigate to the project directory:

   ```bash
   cd Lembah-Peladang-Agro-Park

3. Install dependencies:
    
    ```
    cd lembah-peladang-agro-park
    npm install 
    cd..
    cd server
    npm install
    ```
4. Setup the .env in the Server directory

5. Start the environment: 

    Frontend
    ```
    npm run dev
    ```
    Server
    ```
    npm start
